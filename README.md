# Diary Day App

```This application can create your story public and private.```
### Tech Stacks
NodeJS, ExpressJS, MongoDB, Handlebars (HBS), Google OAuth 2.0 with Passport, Materialize. 

### Running This App
- copy ```./config/config.env.example``` to ```./config/config.env```
- fill the ```config.env``` file with the following your MongoDB URI, Google Client ID, and Google Client Secret.
- run ```npm install```
- run ```npm run dev``` for development and ```npm start``` for production.
